快捷键可以自定义

快捷键的配置存在于数据文件夹中的 `keybinds.json` 文件中，如果改乱了，可以直接删除该文件，然后重启软件即可。

快捷键甚至可以监听鼠标事件和鼠标滚轮

## 快捷键的格式

快捷键序列由多个按键组成，使用空格分隔，代表**依次**按下每个按键

每个按键由“修饰键+按键”组成，用连字符“-”分隔。

| 修饰键 | 代表的按键 | Windows | macOS   | Linux (默认 keymap) |
| ------ | ---------- | ------- | ------- | ------------------- |
| C      | Ctrl       | Ctrl    | Control | Ctrl                |
| A      | Alt        | Alt     | Option  | Alt                 |
| S      | Shift      | Shift   | Shift   | Shift               |
| M      | Meta       | Windows | Command | Super               |

| 特殊按键 | 代表的按键   |
| -------- | ------------ |
| `<MWU>`  | 鼠标滚轮向上 |
| `<MWD>`  | 鼠标滚轮向下 |
| `<x>`    | 鼠标按键 x   |
