## 让模型生成图

模型需要支持联网

在提示词开头插入 `https://import.project-graph.top` 即可自动插入相关提示词

例如:

```
https://import.project-graph.top
帮我随便生成一个人物关系图
```

## mermaid 语法

在 2.4.2+ 版本中，project-graph 已经支持 mermaid 语法，可以直接把 框嵌套网状结构导出为 mermaid 复制到粘贴板中，发送给大模型。

以及让大语言模型生成 mermaid 图，导入到 project-graph 中（在顶部导航栏中）。
