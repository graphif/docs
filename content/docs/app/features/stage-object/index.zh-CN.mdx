舞台对象分为两大类：实体类型、关系类型

舞台对象表示一切在舞台上出现的且被持久化保存在文件中的对象。

临时产生的特效对象、吸附在鼠标上的框选矩形对象、等，不会存储在文件中，而是临时生成并在场景中显示。
