## Introduction

Project Graph is a software that focuses on quickly drawing diagrams. It aims to help you brainstorming more efficiently. You can create and connect nodes by simply dragging them.

[CLICK HERE](/release/latest)

### AUR

If you are using Arch Linux, you can install Project Graph from AUR.

- [project-graph-nightly-bin](https://aur.archlinux.org/packages/project-graph-nightly-bin)
- [project-graph-bin](https://aur.archlinux.org/packages/project-graph-bin)
- [project-graph-git](https://aur.archlinux.org/packages/project-graph-git)
