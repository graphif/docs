## History

From Sep 21, 2024 to Nov 16, 2024, the project has been converted from PyQt to Tauri, and all functions have been implemented gradually, then version 1.0.0 was released.

| 重要版本号 | 重要内容与版本号含义                 |
| ---------- | ------------------------------------ |
| 1.0.0      | All features in the PyQt version     |
| 1.1.0      | Auto compute engine                  |
| 1.2.0      | Image node                           |
| 1.3.0      | Line breaks in text node             |
| 1.4.0      | Colors on edges                      |
| 1.4.20     | Pen strokes                          |
| 1.4.40     | Custom width of text node            |
| 1.5.0      | Remove bundled MiSans font           |
| 1.6.0      | Adjustable connection point position |
| 1.7.0      | MTU (multi target undirected) Edge   |
| 1.8.0      | Multi window                         |
| 2.0.0      | Tabs                                 |
