如何在本地运行项目？

## 手动搭建开发环境

### 环境要求

| Rust    | Node.js            | pnpm    |
| ------- | ------------------ | ------- |
| ^1.81.0 | ^20.19.0 \| ^22.12 | ^10.6.5 |

### 克隆仓库

```sh
git clone https://github.com/LiRenTech/project-graph.git
```

### 安装依赖

```sh
pnpm install
```

### 启动项目

```sh
pnpm tauri:dev
```

```
xlings install
```
