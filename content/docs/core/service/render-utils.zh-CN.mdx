RenderUtils 服务用于在画布上渲染基础图形。它提供了在 View 坐标系下绘制图形的方法。

## API

### `renderPixel(location: Vector, color: Color)`

绘制一个像素点。

### `renderArrow(direction: Vector, location: Vector, color: Color, size: number)`

绘制一个箭头（只绘制箭头头部，不绘制线条）。
