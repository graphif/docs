此服务专注于在应用程序的用户界面中，以视觉方式突出显示搜索操作所发现的内容。它通过动态渲染效果，确保用户能够清晰地识别出搜索结果在视图中的位置。

## API 方法

### `render(frameTickIndex: number)`

根据当前的帧计时索引，绘制并更新所有搜索结果的高亮显示。此方法会为每个搜索结果节点生成一个带有脉冲动画效果的矩形阴影，以提供直观且引人注目的视觉反馈。
