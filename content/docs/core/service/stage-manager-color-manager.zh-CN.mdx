用于管理项目舞台上各种对象的颜色。它允许用户对选中的节点和连线进行颜色设置、变暗或变亮操作，并记录这些操作以便于历史回溯。

## API

### `setSelectedStageObjectColor(color: Color)`

将所有当前选中的舞台对象（包括文本节点、区域、笔画、SVG节点和关联线）的颜色设置为指定颜色。

### `darkenNodeColor()`

将所有当前选中的文本节点的颜色调暗。

### `lightenNodeColor()`

将所有当前选中的文本节点的颜色调亮。
