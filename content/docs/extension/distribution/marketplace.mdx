title: 扩展市场
icon: Container
---

推荐使用官方的[扩展市场](https://hub.graphif.dev/ext/marketplace)来分发扩展。

如果想要你的扩展在扩展市场上出现，需同时满足以下所有条件：

- 扩展必须在 GitHub 上开源，并且必须是 Public 仓库
- 仓库必须有 `extprg` Topic
- 默认分支的根目录必须包含 `metadata.msgpack` 或由 `create-extprg` 生成的 `package.json`，并且内容符合相关规范
- 仓库需要有至少一个 Release，并且 Release 中包含一个 `.prg` 文件

服务器会定期扫描 GitHub 上的仓库，并自动将符合条件的扩展添加到扩展市场中，请耐心等待至少 5 分钟。
