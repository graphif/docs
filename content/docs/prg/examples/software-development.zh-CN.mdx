## 快速跳转代码位置

对于程序员来说，最大的恐惧莫过于突然接手一个大的屎山项目，往往一个功能点需要跳转很多文件中的很多片段才能梳理完整。

在VSCode插件市场中可以搜索 “file ref tags” 插件，认准发布者为“LiRenTech”。（它是由阿岳开发的VSCode插件）

剩余具体详见 **特性 - 集成 - VSCode** 章节


<Callout type="warning">
（由于网站url结构可能随时发生更改，此处暂不做可点击的跳转链接，需要麻烦您自行在侧边栏中查找）
</Callout>

## 自动生成文件夹嵌套结构

可以选中一个文件夹，生成它的框框嵌套图。

详见b站上的 2.0 视频

<Callout type="warning">
目前还没有识别 .gitignore 文件，一定要确保你的文件夹是干净的，是刚 git clone 的文件夹。

否则 node_modules 或者 dist 之类的文件夹 会被识别，里面成千上万的文件会被遍历，巨卡无比
</Callout>
