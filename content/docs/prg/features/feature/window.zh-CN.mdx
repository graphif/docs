软件的窗口目前有一些特殊功能：

- 调整舞台透明度
- 开启狙击镜
- 开启穿透点击
- 置于顶层
- 呼出软件窗口

## 舞台背景透明度

可以在设置中调整舞台背景透明度，也可以用快捷键小幅度调整和快速切换完全透明与完全不透明。

此功能可以用于层叠画画临摹、层叠窗口一边看视频一边做记录等……

<Callout>
为什么软件第一次下载时，默认会时0.8的不透明度？而不是完全不透明？

这是为了防止您不知道这个功能的存在。所以才刻意设置的，您想要调整透明度，就会有意识的寻找设置项界面，进而自然而然的熟悉了这个软件。

</Callout>

- 增加不透明度：`Ctrl+Alt+Shift+⬆️`
- 减少不透明度：`Ctrl+Alt+Shift+⬇️`
- 切换不透明度为 0%或 100%：`Ctrl+0`

## 穿透点击

开启穿透点击后，窗口会自动置于顶层。舞台透明度会自动降低。并且窗口会变成只能观看，无法操作点击到的状态，因此只能通过全局快捷键 `Alt + 2` 来关闭这种状态。

此全局快捷键可以在设置中禁用。

## 呼出软件窗口

当软件最小化时，按 `Alt+1` 全局快捷键可以呼出窗口。（此全局快捷键可以在设置中禁用。）

## 置于顶层

windows 点击右上角 📌 图钉按钮可以将窗口置于顶层。mac 点击左上角红黄绿蓝中的蓝色按钮顶置窗口。

## 狙击镜

狙击镜的功能来自于实验。可以用于防窥、卡片遮挡法记忆训练。

开启狙击镜后，只会渲染显示鼠标附近的圆形区域。可以和舞台背景透明一起开启，以增强防窥效果。

开启狙击镜后，Shift+鼠标滚轮可以调整狙击镜半径大小。
