## 核心理念

<Cards>
  <Card icon={<Telescope />} title="「极简」">
    屏幕上不应出现过多的元素，保持界面简洁、美观，让用户专注于画布
  </Card>
  <Card icon={<Layers />} title="「层叠」" href="/docs/ui/layers">
    使用不同的层级来区分弹出「画布」「窗口」「对话框」等内容，点击此卡片了解更多
  </Card>
  <Card icon={<Brush />} title="「定制」">
    得益于 shadcn-ui 强大的主题系统，用户可以轻松地编写 `.pg-theme` 主题文件并导入应用
  </Card>
</Cards>

## 组件库

使用可丰富自定义的 [`shadcn-ui`](https://ui.shadcn.com/) 作为组件库
