---
title: 支持的 URI 协议
icon: Link
---

文件协议指 `schema://user:password@host:port/path?query#fragment` 中的 `schema` 部分，Project Graph 支持多种 URI 协议来访问和管理文件。

| 功能                  | `file` | `draft`               | `http/https`                               | `http/https` (支持 WebDAV 协议) |
| --------------------- | ------ | --------------------- | ------------------------------------------ | ------------------------------- |
| 读取文件              | ✅     | 🎭 始终返回空白文件   | ✅                                         | ✅                              |
| 读取目录              | ✅     | ❌                    | 🎭 仅支持 Nginx 且 `autoindex_format json` | ✅                              |
| 写入文件              | ✅     | 🎭 弹出保存文件对话框 | ✅                                         | ✅                              |
| 删除文件              | ✅     | ❌                    | ✅                                         | ✅                              |
| 检查文件/目录是否存在 | ✅     | 🎭 始终返回 `false`   | ✅                                         | ✅                              |
| 创建目录              | ✅     | ❌                    | ❌                                         | ✅                              |
| 移动/重命名文件或目录 | ✅     | ❌                    | 🎭 先删除文件再上传文件                    | ✅                              |
