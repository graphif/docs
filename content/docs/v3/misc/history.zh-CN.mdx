## 版本历史

### 1.0 时代

从 2024 年 9 月 21 日 至 11 月 16 日，Project Graph 从 PyQt 版本转成 Tauri 版本，逐渐实现全部功能，总共用时 56 天。1.0.0 正式发布

| 重要版本号 | 重要内容与版本号含义                      |
| ---------- | ----------------------------------------- |
| 1.0.0      | 完成 Pyqt 版本的全部功能，增加 Section 框 |
| 1.1.0      | 逻辑节点的自动计算引擎                    |
| 1.2.0      | 图片节点                                  |
| 1.3.0      | 文本节点：支持输入换行符换行              |
| 1.4.0      | 连线：设置颜色                            |
| 1.4.20     | 涂鸦                                      |
| 1.4.40     | 文本节点：调整宽度强制换行（ttt）         |
| 1.5.0      | 移除 MiSans 字体，安装包体积大幅度减小    |
| 1.6.0      | 连线端点位置可以调整，贝塞尔曲线优化      |
| 1.7.0      | 增加无向边/凸包                           |
| 1.8.0      | 增加子窗口                                |

若要查看详细的版本内容，请访问 [GitHub Releases](https://github.com/LiRenTech/project-graph/releases)

### 2.0 时代

自 2.1 版本开始，项目进入 v2 与 v3 同时开发的状态，阿岳主要负责开发 v2，zty 主要负责开发 v3，最终 v3 版本会取代 v2

| 重要版本号 | 重要内容与版本号含义                                |
| ---------- | --------------------------------------------------- |
| 2.0.0      | 增加 prg 独立文件格式、多标签页、快捷键与秘籍键合并 |
| 2.1.0      | 基本完成 1.8~2.0 版本的重构、补全功能遗漏           |
