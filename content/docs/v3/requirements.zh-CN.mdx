title: 系统要求
icon: Cable
---

## Windows

- 操作系统: Windows 10 (`x86_64`) 或更高版本
- 存储: 至少 20MB 可用空间
- 内存: 至少 150M 可用内存
- `Microsoft Edge WebView2`: `114` 或更高版本

## Linux

- 架构: `x86_64`
- 发行版: 任何基于 glibc 的发行版 (仅 Arch Linux 受官方支持)
- 存储: `/` 挂载点至少 15MB 可用空间，`/home` 挂载点至少 5MB 可用空间
- 内存: 至少 150M 可用内存
- `webkit2gtk-4.1`: `2.50.0` 或更高版本
- 显示协议: X11 或 Wayland (仅 Wayland 受官方支持)

## macOS

- 架构: `aarch64` 或 `x86_64` (仅 `aarch64` 受官方支持)
- 操作系统: macOS 14.0 (Sonoma) 或更高版本
- 存储: 至少 20MB 可用空间
- 内存: 至少 150M 可用内存
- Safari 浏览器: `16.4` 或更高版本

## 网页版

Chrome 114, Safari 16.4, 不支持 Firefox
